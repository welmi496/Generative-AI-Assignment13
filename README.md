# Assignment 13: Generative AI Essentials

## Project Overview
This project demonstrates the basics of Generative AI through a simple text generation model built in Google Colab using TensorFlow/Keras. The notebook uses a public-domain text dataset from Project Gutenberg and trains a character-level model to generate text from a seed prompt.
